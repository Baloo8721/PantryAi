# FreshCart - AI Meal & Deal Planner

## Project Overview
- **Type**: Single-page web application (HTML/CSS/JS)
- **Core functionality**: Smart grocery/meal planning with deal-finding and AI assistant
- **Target users**: Home cooks wanting to optimize meal planning and grocery savings

## Features
- Fridge inventory management
- AI-powered meal planning (Mediterranean + custom diets)
- Smart shopping list with deal matching
- Store deals radar (Publix, Aldi, Greenwise, Amazon)
- Coupon stacking simulation
- OpenRouter AI chat assistant
- localStorage persistence (no backend required)

## Deployment
- Netlify or GitHub Pages - no build step required
